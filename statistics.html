Key Formulas in Statistics

1. Mean (Average)

Formula:
\( \text{Mean} = \frac{x_1 + x_2 + \ldots + x_n}{n} \)

Example:
For the data set \( 4, 8, 6, 5, 3 \):
\( \text{Mean} = \frac{4 + 8 + 6 + 5 + 3}{5} = \frac{26}{5} = 5.2 \)

2. Median

Formula:
If \( n \) is odd: \( \text{Median} = x_{\frac{n+1}{2}} \)
If \( n \) is even: \( \text{Median} = \frac{x_{\frac{n}{2}} + x_{\frac{n}{2} + 1}}{2} \)

Example:
For the data set \( 3, 5, 7, 9, 11 \) (odd):
\( \text{Median} = 7 \)
For the data set \( 3, 5, 7, 9 \) (even):
\( \text{Median} = \frac{5 + 7}{2} = 6 \)

3. Mode

Formula:
The mode is the number that appears most frequently in a data set.

Example:
For the data set \( 1, 2, 2, 3, 4 \):
The mode is \( 2 \).

4. Range

Formula:
\( \text{Range} = \text{Maximum} - \text{Minimum} \)

Example:
For the data set \( 2, 5, 1, 8 \):
\( \text{Range} = 8 - 1 = 7 \)

5. Variance

Formula:
\( \text{Variance} = \frac{\sum (x_i - \text{Mean})^2}{n} \) (Population Variance)
\( \text{Variance} = \frac{\sum (x_i - \text{Mean})^2}{n-1} \) (Sample Variance)

Example:
For the data set \( 4, 8, 6, 5, 3 \):
Mean = \( 5.2 \)
Variance = \( \frac{(4-5.2)^2 + (8-5.2)^2 + (6-5.2)^2 + (5-5.2)^2 + (3-5.2)^2}{5} = \frac{1.44 + 7.84 + 0.64 + 0.04 + 4.84}{5} = 2.80 \)

6. Standard Deviation

Formula:
\( \text{Standard Deviation} = \sqrt{\text{Variance}} \)

Example:
Continuing from the previous variance example:
\( \text{Standard Deviation} = \sqrt{2.80} \approx 1.67 \)

7. Probability

Formula:
\( P(A) = \frac{\text{Number of favorable outcomes}}{\text{Total number of outcomes}} \)

Example:
The probability of rolling a 3 on a die:
\( P(3) = \frac{1}{6} \)

8. Combinations

Formula:
\( C(n, r) = \frac{n!}{r!(n-r)!} \)

Example:
The number of ways to choose 2 items from 5:
\( C(5, 2) = \frac{5!}{2!(5-2)!} = \frac{5 \times 4}{2 \times 1} = 10 \)

9. Permutations

Formula:
\( P(n, r) = \frac{n!}{(n-r)!} \)

Example:
The number of ways to arrange 3 items from 5:
\( P(5, 3) = \frac{5!}{(5-3)!} = \frac{5 \times 4 \times 3}{1} = 60 \)

10. Z-Score

Formula:
\( Z = \frac{X - \text{Mean}}{\text{Standard Deviation}} \)

Example:
If \( X = 10 \), Mean = 8, and Standard Deviation = 2:
\( Z = \frac{10 - 8}{2} = 1 \)
