Matrix Formulas

Matrix Addition

If A and B are two matrices of the same dimension, then their sum C is:

\[ C = A + B \]

Where each element of C is given by:

\[ C_{ij} = A_{ij} + B_{ij} \]

Matrix Subtraction

If A and B are two matrices of the same dimension, then their difference C is:

\[ C = A - B \]

Where each element of C is given by:

\[ C_{ij} = A_{ij} - B_{ij} \]

Scalar Multiplication

If A is a matrix and k is a scalar, then the scalar multiplication B is:

\[ B = kA \]

Where each element of B is given by:

\[ B_{ij} = k \cdot A_{ij} \]

Matrix Multiplication

If A is an m × n matrix and B is an n × p matrix, their product C is:

\[ C = AB \]

Where each element of C is given by:

\[ C_{ij} = \sum_{k=1}^{n} A_{ik} \cdot B_{kj} \]

Transpose of a Matrix

The transpose of a matrix A, denoted by AT, is obtained by swapping rows and columns:

\[ A^T = (A_{ij})^T = A_{ji} \]

Determinant of a 2x2 Matrix

If A is a 2x2 matrix:

\[ A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \]

The determinant of A is given by:

\[ \text{det}(A) = ad - bc \]

Inverse of a 2x2 Matrix

If A is a 2x2 matrix with a non-zero determinant:

\[ A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \]

The inverse of A is given by:

\[ A^{-1} = \frac{1}{ad - bc} \begin{pmatrix} d & -b \\ -c & a \end{pmatrix} \]

Identity Matrix

An identity matrix I is a square matrix in which all the elements of the principal diagonal are ones and all other elements are zeros:

\[ I = \begin{pmatrix} 1 & 0 & \dots & 0 \\ 0 & 1 & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & 1 \end{pmatrix} \]
