Modulo Formulas

1. Modulo Operation

The modulo operation finds the remainder when one integer is divided by another. It is expressed as:

a % b = r

Where:

  • a is the dividend
  • b is the divisor
  • r is the remainder

2. Properties of Modulo

(a + b) % c = [(a % c) + (b % c)] % c

This property is used to find the modulo of a sum.

(a - b) % c = [(a % c) - (b % c) + c] % c

This property is used to find the modulo of a difference.

(a * b) % c = [(a % c) * (b % c)] % c

This property is used to find the modulo of a product.

3. Modulo Inverse

The modulo inverse of an integer a with respect to modulus m is an integer x such that:

(a * x) % m = 1

This only exists if a and m are coprime.

4. Modulo Exponentiation

To calculate (ab) % c efficiently, we use modulo exponentiation:

(ab) % c = [(a % c)b % c]

5. Division under Modulo

To divide two numbers under modulo m, multiply the dividend by the modulo inverse of the divisor:

(a / b) % m = [(a % m) * (b-1 % m)] % m

Where b-1 is the modulo inverse of b under m.
