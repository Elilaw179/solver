Mapping Formulas

1. Function Definition

A function \( f \) is a mapping from set \( A \) to set \( B \) such that every element of \( A \) is related to exactly one element of \( B \).

\( f: A \rightarrow B \)

2. Domain and Range

Domain: The set of all possible input values (elements of set \( A \)) for the function.

Range: The set of all possible output values (elements of set \( B \)) that the function can produce.

3. One-to-One Function (Injective)

A function \( f \) is injective (one-to-one) if different elements in \( A \) map to different elements in \( B \).

\( f(a_1) \neq f(a_2) \) for \( a_1 \neq a_2 \)

4. Onto Function (Surjective)

A function \( f \) is surjective (onto) if every element of set \( B \) is the image of at least one element of set \( A \).

\( \text{For all } b \in B, \text{ there exists } a \in A \text{ such that } f(a) = b \)

5. Bijective Function

A function is bijective if it is both injective and surjective.

\( f: A \rightarrow B \) is bijective if for every element \( b \) in \( B \), there is a unique element \( a \) in \( A \) such that \( f(a) = b \).

6. Identity Function

The identity function \( I \) maps every element of set \( A \) to itself.

\( I(a) = a \text{ for all } a \in A \)

7. Composite Function

The composite function \( g \circ f \) is formed by applying function \( f \) first, and then applying function \( g \) to the result.

\( (g \circ f)(x) = g(f(x)) \)
